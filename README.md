# Mwena Agritech

Mwena Agritech is a Python project designed to help farmers analyze their sales data and make better decisions to increase product revenue. Using simple data analysis and visualizations, farmers can understand which products sell best, when, and how to improve their business.

## Features

- Upload and analyze farm sales data (CSV format)
- Visualize sales trends, product revenue, and distribution
- Download and organize agricultural images respectfully (Ubuntu Image Fetcher)
- Educational demos included for learning purposes
